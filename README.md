# Stress

Implementation of the card game Stress:
https://sv.wikipedia.org/wiki/Stress_(kortspel)

Includes
- Game model in C#
- WPF test client
- Web server/client with 2 player support over SignalR

Live at: https://granakerprod-stress.azurewebsites.net/
